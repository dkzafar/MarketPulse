Comprehensive All-In-One Prompt for Building Your Next-Gen Stock Analysis Dashboard

Context:
You’re building a web-based stock analysis dashboard with a professional, Netflix/Prime-inspired dark-mode UI that rivals TradingView and Yahoo Finance. Data comes from Yahoo Finance’s public APIs and is enhanced with AI-generated insights, micro-copy, and animations using free online AI tools. The site must be fully interactive, mobile-responsive, and packed with advanced features to keep users engaged and informed.

⸻

1. Core Functionality
	1.	Watchlist & Search
	•	A persistent left sidebar with:
• Logo and site title in a bold accent color.
• A search bar that auto-suggests symbols (AAPL, TSLA, etc.)
• A “Favorites” watchlist you can add/remove symbols from.
• Navigation links: Summary, Charts, Indicators, News, Settings.
	2.	Summary Table
	•	A horizontal, scrollable carousel of Summary Cards for each watchlist symbol showing:
• Symbol + company name
• Current price, daily change % (animated green/red)
• Market cap, P/E ratio, dividend yield.
	3.	Interactive Historical Price Chart
	•	A full-width, responsive line chart with zoom & pan.
	•	X-axis = date, Y-axis = price.
	•	Tooltip on hover showing exact values.
	•	Range selector (1M, 6M, 1Y, 5Y).
	4.	Financial Indicators Panel
	•	A grid of mini-charts for RSI, MACD, Volume, Bollinger Bands.
	•	2×2 layout on desktop, stacked on mobile.
	5.	News & Events Feed
	•	A vertical list of the latest headlines for the selected symbol.
	•	Each item shows timestamp, source, sentiment icon.
	•	Clicking opens the original article.
	6.	AI Insights Section
	•	Below the Indicators panel, an embeddable “AI Insights” card:
• Summarizes key patterns (e.g. “RSI is near 70, suggesting overbought conditions.”)
• Generated via ChatGPT using the fetched indicator data.
	7.	User Settings & Profile
	•	Toggle dark/light mode.
	•	Manage watchlist persistence in localStorage.
	•	Select theme accents (Neo-Noir, Midnight Blue, Carbon Black).
	•	Configure price alerts with browser push notifications.
	•	View referral code and invite friends for bonus features.

⸻

2. Visual & Interaction Design
	1.	Overall Aesthetic
	•	Background: #0B0D17 (true black)
	•	Surfaces: #1E2128 (charcoal)
	•	Accent: #E50914 (Netflix red) with secondary #FFAD1F (Prime gold)
	•	Text: #FFFFFF (primary), #B0B0B0 (secondary)
	•	Fonts: Montserrat (400/600/800)
	•	Spacing: 4px–64px scale, consistent 8px baseline
	•	Borders: 8px radius on cards/buttons
	2.	Layout (Netflix-Style Rows)
	•	Fixed sidebar (desktop) collapsing to a hamburger menu on mobile
	•	Main area with horizontal “rows”:
• Row 1: Summary Cards carousel
• Row 2: Historical Price Chart (full width)
• Row 3: Indicators grid
• Row 4: AI Insights card + News feed side by side
	3.	Animations & Micro-Interactions
	•	Framer-Motion slide-in for each row on load, staggered by 100ms
	•	Summary cards scale up (1.03×) with glow on hover/focus
	•	Price change values animate color and count up/down
	•	Chart lines draw with easing over 1s on mount
	•	Tooltip fade in/out smoothly
	•	Row hover and click ripples using Lottie micro-animations

⸻

3. Data & Performance
	1.	Data Fetching
	•	Use React-Query (TanStack) hooks:
• useQuotes(symbols[]) for real-time quotes (staleTime: 5m)
• useHistory(symbol, range) for historical data
• useIndicators(symbol) for RSI, MACD, etc.
	•	Cache data client-side, refetch on focus/reconnect.
	2.	Offline & PWA
	•	Enable Service Worker with Workbox for offline caching of the last-fetched menu, charts, and AI insights.
	•	Provide an “Add to Home Screen” prompt after two visits.
	3.	Responsive
	•	Sidebar collapses to top nav on < 768px
	•	Rows stack vertically on < 600px
	•	Charts reflow to full-width containers on mobile

⸻

4. AI Tools & Prompts (No Code Needed)
	1.	ChatGPT (chat.openai.com)
	•	AI Insights Copy:
“Summarize these RSI and MACD values for SYMBOL over the past month in two clear sentences, with a confident, trader-friendly tone.”
	•	Microcopy & Tooltips:
“Write friendly, concise tooltips for a dark-mode finance dashboard: search bar auto-suggest, summary card price change, chart zoom, RSI indicator, settings toggle.”
	2.	DALL·E 3 (labs.openai.com)
	•	Header Graphic:
“Create a dark, cinematic hero background of neon data streams converging into a glowing node with a subtle Netflix-red accent.”
	•	Section Illustrations:
“Generate a minimal line-art pattern of finance icons (candlesticks, pie charts, pizza slice easter egg) in dark mode.”
	3.	Stable Diffusion (DreamStudio Trial)
	•	Backdrop Textures:
“Design a dark grid/circuit texture with occasional red-orange pulses for a financial dashboard background.”
	4.	LottieFiles
	•	Micro-animations — search or create:
	1.	Chart draw animation
	2.	Button hover glow
	3.	Tooltip fade pop
	4.	Confetti burst
	5.	Colormind
	•	Palette Expansion:
“Given black (#0B0D17), charcoal (#1E2128), Netflix-red (#E50914), Prime-gold (#FFAD1F), white (#FFFFFF), suggest 4 complementary accent tones and usage recommendations.”
	6.	Khroma
	•	Font Pairings:
“Recommend three Google Font pairings for Montserrat that work well in a dark, data-focused UI.”

⸻

5. Upgrades & Enhancements
	1.	Real-Time WebSocket Feed
“Integrate free IEX Cloud WebSocket for live tick updates; animate summary cards in real time.”
	2.	AI-Powered News Summaries
“Fetch top 5 news headlines for SYMBOL via NewsAPI.org, then prompt ChatGPT: ‘Summarize each headline in one sentence with sentiment.’”
	3.	Voice Commands
“Use the Web Speech API: when user says ‘Show me Tesla’, navigate to TSLA charts.”
	4.	Shareable Chart Snapshots
“Generate Open Graph images on-the-fly via DALL·E: embed SYMBOL, current price, and a small sparkline on a dark background.”
	5.	Portfolio Tracker
“Allow users to input holdings; display P&L and allocation pie chart; compare vs. benchmarks.”
	6.	Alert Engine
“Let users set price alerts; trigger browser push notifications via free OneSignal plan when thresholds are crossed.”
	7.	Backtest Simulation
“Enable users to simulate buying SYMBOL at past dates; calculate ROI and display results in a chart and summary card.”
	8.	A/B Testing & Analytics
“Set up Google Optimize to test two hero backgrounds or CTA texts; track conversions via Google Analytics.”
	9.	Accessibility Audit
“Run axe-core in CI; fix all ARIA, focus, and contrast issues to meet WCAG AA.”
	10.	User Onboarding Gamification
“Embed a Typeform quiz in Settings that recommends a watchlist based on risk profile; award bonus points for completion.”

⸻

How to Execute:
	1.	Copy-Paste this entire prompt into your AI assistant (Replit AI, ChatGPT, etc.).
	2.	Run stage by stage or all at once, depending on your tool’s capabilities.
	3.	Review & Integrate the generated assets, styles, copy, and configurations into your Next.js codebase.
	4.	Test across devices, browsers, and Lighthouse to hit >90 in all categories.

This single, unified prompt covers every aspect — from data hooks and layout to AI-powered microcopy, animations, and advanced features — using entirely free AI tools online. It will guide the AI to produce a cohesive, interactive, and visually stunning stock dashboard that matches your brand vision.
